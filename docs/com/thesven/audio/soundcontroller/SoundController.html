createSoundGroup()method 
public function createSoundGroup(sndGrpName:String, groupItems:Vector):Boolean

Used to create a new sound group

In order to create a sound group all of the sounds must all ready be added using addEmbededOrLibrarySound or addExternalSound

Using a sound group you can manage the state of one or more sounds by using a single call ie. play, stop, pause

Individual sounds may be used in more than one group

Dispatches the GROUP_ADDED DataEvent with a value of "sndGroupName"

Parameters
sndGrpName:String — - the name you wish to use for the sound group
 
groupItems:Vector — - a vector containing the names of the sound objects to be used in the sound group

Returns

public function playSound(sndName:String, useSndProps:Boolean = true, volume:Number = 1, startTime:Number = 0, loops:int = 0, useDelay:Boolean = false, delayInSeconds:Number = 0):void

Used to play a sound that has been added to the controller

Parameters
sndName:String — - the name of the sound you wish to play
 
useSndProps:Boolean (default = true) — - would you like to use the current properties associated with the sound, or supply new ones through the method parameters
 
volume:Number (default = 1) — - if useSoundProps = true, this new volume will be used when playing the sound
 
startTime:Number (default = 0) — - if useSoundProps = true, this new startTime will be used when playing the sound
 
loops:int (default = 0) — - if useSoundProps = true, this amount will over ride the current amount of loops that the sound plays
 
useDelay:Boolean (default = false) — - if true a delay will be used before playing the sound
 
delayInSeconds:Number (default = 0) — - if useDelay = true, the sound will be delayed "x" seconds before playing
